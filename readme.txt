A minimal plugin that changes the alt-text of the product images, according to the product title.

== Description ==

 Bulk Alt-Text Changer for SEO is a minimal yet powerful tool designed to enhance the accessibility and SEO of your WordPress website effortlessly.
 With a focus on simplicity and effectiveness, Alt Text Sync automates the process of updating alt text for product images based on their corresponding product titles.
