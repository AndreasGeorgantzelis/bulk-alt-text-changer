This plugin changes the alt-text of the product images, according to the product title.

== Description ==

Bulk Alt-Text Changer for SEO is a convenient solution tailored for managing alt text for product images with ease.
Designed to enhance both accessibility and search engine optimization (SEO), this plugin automates the process of aligning alt text with product titles.
